# minikube

<https://kubernetes.io/zh/docs/setup/learning-environment/minikube/>

minikube 能快速创建k8s的开发集群，支持在虚拟机上创建，也支持裸机创建.

    // 在裸机上创建：
    sudo minikube start --vm-driver=none
