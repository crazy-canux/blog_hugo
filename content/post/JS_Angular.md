install angular-cli:

    $ npm install -g @angular/cli
    
verify:

    $ ng v
    
新建项目:

    $ ng new <my-app>
    
测试项目:

    $ cd <my-app>
    // --open会自动打开浏览器
    $ ng serve --open
    
打包

    $ ng build --prod
