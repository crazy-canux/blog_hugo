# Map-Reduce

一种基于YARN的大型数据并行处理系统．主要处理离线数据．

***

# mapreduce commands

    mapred [SHELL_OPTIONS] COMMAND [GENERIC_OPTIONS] [COMMAND_OPTIONS]

user commands:

    $ mapred archive
    ...

admin commands:

    $ mapred historyserver

    $ mapred hsadmin
